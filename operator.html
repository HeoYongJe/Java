복합 대입 연산자

+= 더한 후 대입
-= 뺀 후 대입
*= 곱한 후 대입
/= 나눈 후 대입
%= 나머지를 구한 후 대입
a = 5;
a += 5; 결과 : a = 10
a = 5;
a -= 5; 결과 : a = 0
a = 5;
a *= 5; 결과 : a = 25
a = 5;
a /= 5; 결과 : a = 1
a = 5;
a %= 5; 결과 : a = 0
++ 1을 더한다
-- 1을 뺀다
  • a = 5;
  • b = 1;
  • a = b++ 결과 : a = 5;
  • a = ++b 결과 : a = 6;
  • 증감은 앞 뒤 순서에 따라 값이 바뀐다
&& 좌변과 우변이 모두 true인 경우 true
|| 좌변과 우변 둘중 하나 이상이 true인 경우 true
! 해당 변수가 true인 경우 false, false인 경우 true ( !변수명 )
