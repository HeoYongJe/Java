자바기초

  • 변수
  • 자바에서는 변수에 값이 저장되지 않으면 변수가 생성되지 않는다
  • 변수의 선언과 생성은 다르다
    • int score; //선언 : 저장되는 값의 종류와 이름만 언급
    • score = 90; //값 저장: 변수에 최초로 값이 저장될 때(초기값) 생성된다, 변수초기화라고도 한다
  • int score = 90; // 초기값은 변수를 선언함과 동시에 제공할 수도 있다
  • 초기화되지 않은 변수는 아직 메모리 번지 정보를 가지고 있지 않으므로 변수를 통해 메모리 값을 읽을 수 없다
    • int value; //변수선언
    • int result = value + 10; //선언은 했지만 초기화가 안됬기 때문에 value + 10에서 value변수는 존재하지 않는다
