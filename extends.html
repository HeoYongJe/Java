상속

반복되는 작업을 피하기 위해

  • ex) public class Human(){
    먹기
    잠자기
    씻기
    }
  • public class Father() extends Human(){
    일하기
    }
  • public class Mother() extends Human(){
    요리하기
    }
  • public class Son() extends Human(){
    공부하기
    }

위처럼 extends Human 통해 동일한 작업을 반복하지 않고 상속한다

하지만 먹기, 잠자기, 씻기처럼 반복이지만 자식 클래스에서 세부적인 내용이 다를 수 있다.

ex)먹기 이지만 많이먹기, 적게먹기, 매우 적게 먹기 등

이런경우 override 메소드를 사용 할 수 있다
public class Son() extends Human(){
공부하기
override 먹기 {매우 적게 먹기}
}
